# MS2 WiFi Digital Microscope — Quick Reference Card

## Tombol Cepat (Shortcut)

| Aksi | Cara |
|---|---|
| Nyalakan | Long press Power |
| Matikan | Long press Power |
| Aktifkan WiFi | Short press Power |
| Ambil foto | OK (saat mode Photo) |
| Mulai/stop video | OK (saat mode Video) |
| Ganti mode | Long press Menu |
| Masuk/keluar menu | Short press Menu |
| Zoom in | Up button |
| Zoom out | Down button |
| Reset freeze | Tekan Reset hole |

---

## Koneksi Cepat WiFi ke Android

```
1. Nyalakan MS2
2. Short press Power → WiFi icon berkedip
3. HP → Settings → WiFi → "wifi_camera_MS2_XXXX"
4. Connect (abaikan "No Internet")
5. Buka app Max-see → Camera
```

---

## USB Modes saat konek ke PC

```
Memory Mode    → Akses TF card (seperti flashdisk)
PC Camera Mode → Webcam UVC (pakai Windows Camera / Amcap)
Charge Mode    → Isi daya, device tetap jalan normal
```

---

## Troubleshooting Kilat

| Gejala | Fix |
|---|---|
| Gambar buram | Putar Focus Roller |
| App tidak connect | Matikan Mobile Data |
| Device hang | Tekan Reset |
| Baterai baru tidak nyala | Charge dulu sebelum dipakai |
| WiFi tidak muncul | Restart MS2, pastikan tidak dipakai PC |

---

## Spek Singkat

- Video: **1080p FHD** (1920×1080)
- LED: **8 buah** (brightness adjustable)
- Storage: **TF card 8–64GB Class 10**
- Baterai: **18650** (replaceable)
- Fokus: **0–60mm**
- SSID: `wifi_camera_MS2_XXXX`
